# CyberToolkit

CyberToolkit is a Bash-based toolkit for Kali Linux that combines common OSINT and penetration testing utilities into one menu-driven interface.

## Features

- Sherlock username search
- theHarvester email/domain reconnaissance
- WHOIS lookup
- DNS lookup
- IP lookup
- Shodan integration (planned)
- Installed tools checker
- Report generation

## Included tools

- sherlock
- theHarvester
- nmap
- sqlmap
- hydra
- john
- hashcat
- wireshark
- whois
- dig
- curl

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/CyberToolkit.git
cd CyberToolkit
chmod +x install.sh
./install.sh
```

## Run

```bash
~/CyberToolkit/bin/osint
```

## Update

```bash
./update.sh
```
